12 Column Responsive Grid

Each column is a <div> tag with the class column applied to it.

By default, the width of a column is 100%. A custom width can be set by applying a size class. The grid supports 12 column divisions and sizes span a number of 1/12th columns. So a 1x column is 1/12th and a 2x column is 2/12ths and so on.

.no-margin Add the .no-margin class to remove the margin to the right of the column.

.fat Add the .fat class to expand a column to take up margins to both the left and right. In the sample below the .fat class and .no-margin classes alternate
