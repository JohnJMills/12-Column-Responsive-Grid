12 Column Responsive Grid

Each column is a <div> tag with the class column applied to it.

By default, the width of a column is 100%. A custom width can be set by applying a size class. The grid supports 12 column divisions and sizes span a number of 1/12th columns. So a 1x column is 1/12th and a 2x column is 2/12ths and so on.

.no-margin Add the .no-margin class to remove the margin to the right of the column.

.fat Add the .fat class to expand a column to take up margins to both the left and right. In the sample below the .fat class and .no-margin classes alternate

nesting Coulumns can be nested inside each other. To nest columns, first add a child <div> to your column with the .container class. Nested columns should be added insted the .container element.

Be aware that the math for nesting columns doesn't always work well at different sizes. In this example, the last two nested columns are hidden at smaller resolutions because the 8x column is 100% wide on mobile devices while the 1x column is actually 2x wide on mobile devices.

.wide-mobile Some columns may need to expand to 100% wide for mobile but only be 2x or 3x wide in larger resolutions. To force a column to expand to 100% on mobile screens add the .wide-mobile class. Columns in the sample below have the .mobile-wide class applied.

.fixed The grid system is fluid and scales from breakpoint to breakpoint. If you want to fix the widths to the breakpoints (320, 480, 600, 768, 960) then apply the .fixed class to the <body> tag. This will fix the width of the .center-content class to match the breakpoint width.
